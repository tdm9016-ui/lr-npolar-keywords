# lr-npolar-keywords
Extracts geographic keywords from the Norwegian Polar Institute’s map data and exports them as Lightroom‑compatible keyword files.

Download JSON from Npolar here and place it in the same folder as the Python script:
https://next.api.npolar.no/dataset/a2813eb6-e866-4ef7-8808-ed09eb1566c5/attachment/1b805b32-5e9a-4419-a67c-8faee7ab2a25/_blob

Run:
python extract_placenames_LR.py npolar-placenames-api-v1-2020-06-30.json
